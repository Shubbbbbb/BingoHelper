Helpful Recources

Hypixel Skyblock Wiki
The Official Hypixel Skyblock Wiki, maintained by the community. Probably the most expansive archive of information related to Skyblock. I would not have been able to create this without the work the community puts in to maintain this wiki.
Hypixel Forums
An official forum for all things Hypixel related. This is the best place to find information that can't easily be found elsewhere on the internet. Highly reccomend checking here you have a Hypixel related problem.
SkyCrypt
Open source Skyblock stat viewer with tons of useful features. Handy if you want to brag about your digital pixels to your friends. Probably useful for other stuff I guess.

Helpful API's

Hypixel API docs
Official documentation for the Hypixel API. Thourough and easy to understand, great place to start if you want to work with the Hypixel API.
Electroid's Mojang API alternative
An alternative to the Mojant API. I couldn't get the Mojang API to work myself, so this saved me a lot of hassle.
Craftatar API
API that serves png's of Minecraft skins. Absolute godsend for any project that requires the display of a minecraft skin.
